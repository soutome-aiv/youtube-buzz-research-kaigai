# 海外AI情報YouTube研究プロジェクト - プロジェクト計画書

**プロジェクト名：** Claude Code 中心の海外バズ企画リサーチシステム  
**プロジェクトオーナー：** 管理者  
**プロジェクト開始日：** 2026-04-24  
**最終更新：** 2026-04-24

---

## 📌 プロジェクト目的

自社YouTube チャンネル（TOB 経営者向けAI情報発信）において、**最先端のトレンドをキャッチ** し、**マス層に埋もれないユニークな企画** を開発するため、海外（特に US/CA）の Claude Code 関連動画をリアルタイムでリサーチし、企画パターンとバズ要素を自動抽出する。

---

## 🔍 リサーチ対象・スコープ

| 項目 | 内容 |
|------|------|
| **地域** | US/CA（Anthropic 所在地）メイン、Claude Code 流行地域全体 |
| **チャンネル規模** | 大型（>100万登録者）/ 中型（10万～100万）/ 小型（<10万）に分類 |
| **リサーチ対象キーワード** | Claude Code, Claude API, Anthropic, AI開発 |
| **リサーチ要素** | チャンネル企画戦略・個別動画企画・バズ要素・タイムリーなトレンド |
| **ツール** | TubeAI（YouTube 分析 API） |
| **コスト** | 無料 |

---

## 🎯 ゴール

### 最終ゴール
**Google Sheets リアルタイムダッシュボード** を運用し、以下が自動更新される状態

- ✅ Claude Code 関連の全バズチャンネル一覧（規模別）
- ✅ 週単位で更新される「バズ動画ランキング TOP 20」
- ✅ 企画パターン分析（どの企画タイプがバズるのか）
- ✅ 規模別トレンド分布（小型チャンネルと大型チャンネルの企画差異）

### 成功指標
- 初期スキャン完了：関連チャンネル 50～200 件抽出
- 週次更新：毎週金曜 17:00 に新着動画・バズ情報が自動反映
- 企画発見：月 1～2 件の「自社で応用可能な企画パターン」を発見

---

## 🔄 リサーチ方法

### Phase 1: 初期スキャン（準備段階）
```
実施内容：
1. TubeAI search_channels("Claude Code") で関連チャンネル一括検索
2. 登録者数ごとに大型/中型/小型に分類
3. 各チャンネルの基本情報（URL、動画数、最新動画）を取得
4. Google Sheets に「チャンネル一覧」シートとして記録

期待成果：
- 関連チャンネル 50～200 件の リスト化
- 規模別分類完了
```

### Phase 2: 定期リサーチ（運用段階）
```
【毎週金曜 17:00 自動実行】
1. get_channel_videos() で全チャンネルの新着動画取得
2. find_viral_videos() で「バズってる動画（効率スコア >100）」を抽出
3. 動画タイトル・説明文から「企画パターン」を分類
4. Google Sheets の各規模別シートに自動反映

【毎日自動実行】（オプション）
1. get_trending_topics(region="US") で最新 AI トレンド取得
2. 新トレンド登場時はアラート通知
```

### Phase 3: 分析・インサイト抽出（月次）
```
1. 企画パターンの出現頻度分析
2. 規模別での企画差異をレポート化
3. 「今月熱い企画タイプ」を特定
4. 自社応用可能な企画を提案
```

## 📋 出力形式（Google Sheets）

### シート構成

| シート名 | 内容 | 更新頻度 |
|---------|------|---------|
| **チャンネル一覧** | 全チャンネルの基本情報（規模、URL、最新更新日） | 初回 + 月次 |
| **大型チャンネル（>100万）** | 動画タイトル・再生数・企画パターン・投稿日 | 週次 |
| **中型チャンネル（10万～100万）** | （同上） | 週次 |
| **小型チャンネル（<10万）** | （同上） | 週次 |
| **企画パターン分析** | 企画タイプ別の登場頻度・バズ率・規模分布 | 週次 |
| **トレンドアラート** | 新規発見チャンネル・新しい企画パターン | 随時 |
